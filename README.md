# Build Your First MCP Server in Python

A hands-on tutorial for [PyCon 2026](https://us.pycon.org/2026/schedule/presentation/2/) on building MCP servers with Python and FastMCP.

## Slides

https://pamelafox.github.io/pycon2026-mcp-tutorial/

## Exercises

| Exercise | Description |
| --- | --- |
| [Exercise 1](exercise1.md) | Set up your environment, connect a coding agent to public MCP servers, and try an authenticated server |
| [Exercise 2](exercise2.md) | Write a Python agent that uses MCP tools with LangChain, Pydantic AI, or Agent Framework |
| [Exercise 3](exercise3.md) | Build your own MCP server from scratch with FastMCP — a store that lists and sells products |
| [Exercise 4](exercise4.md) | Add visual UI with Prefab charts and cards, plus purchase confirmation with elicitation |
| [Exercise 5](exercise5.md) | Add OAuth 2.1 authentication to your server using Keycloak |
